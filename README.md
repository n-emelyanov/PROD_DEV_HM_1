# GitFlow Workflow
В этом задании необходимо было создать репозиторий, который воспроизводит структуру репозитория по ссылке:



[structure](https://cdn.hashnode.com/res/hashnode/image/upload/v1668069961266/fI6dAXt_8.png) 

![Скриншот](https://cdn.hashnode.com/res/hashnode/image/upload/v1668069961266/fI6dAXt_8.png)

Описание веток, присутствующих в GitFlow:
- **`main/master`** - стабильная версия, готовая к продакшену
- **`develop`** - ветка для разработки, содержит фичи для следующего релиза
- **`feature/*`** - разработка новых функций (ответвляется от develop)
- **`hotfix/*`** - срочные исправления в продакшене (ответвляется от main)
- **`release/*`** - подготовка новой версии к выпуску (ответвляется от develop)

Код воспроизведения структуры репозитория записан в 
[./gitflow.sh](gitflow.sh)
